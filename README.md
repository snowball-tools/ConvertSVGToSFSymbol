# ConvertSVGToSFSymbol

## Getting Started

Convert `SVG` files to `SF Symbol`. Current impl overwrites the original `SVG`

`gem install nokogiri`

### svg file

`ruby SVGToSFSymbol.rb fileName.svg`

### svg file directory

`ruby SVGToSFSymbolDirectory.rb directory`
